This application facilitates the seamless registration of new users, updating a MongoDB database in real time. Built with React.js, the app employs state management to capture user inputs dynamically and display them in a dedicated component at the bottom of the interface. The user experience is enhanced through thoughtful styling with CSS and interactive features powered by JavaScript, ensuring an intuitive and engaging interaction for users.

This application is built using React.js with JavaScript, styled with Tailwind CSS and Flowbite components. It pulls data directly from the RAWG Video Games Database API to dynamically display information on current and trending video games, without the need for a server-side file.

This application was developed using Python, styled with CSS and Bootstrap, and utilizes MySQL as the database. It is a wedding registration app that allows guests to register for the event, manage guest lists, track attendees, and log registered gifts.
